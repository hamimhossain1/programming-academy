Flexbox is one one dimensional layout. It is helpful in allocating and aligning the space among items in a container. Grid is a two-dimensional grid based layout with rows and columns. It makes easier to design web pages without having to use floats and positioning.

The main difference between Bootstrap and Tailwind is that Bootstrap pre-built components that provide a responsive UI components. On the other hand, Tailwind comes with predesigned widgets that facilitate web developers to design a website from scratch.

The CSS Box Model is the primary means of layout for everything on the web. Using the box model, web designers and developers can make use of various properties such as padding, borders, margin-top, margin-right, margin-bottom, margin-left, and many others to create page layouts.

Semantics tag refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. The core characteristic of a semantic element is that it clearly communicated its meaning to both the developer and the browser. These elements clearly define its content. Examples of semantic tags: header, nav, section etc.
